  1. Inline CSS: In this method, we can add the CSS to a HTML element using the style attribute
  2. Internal CSS: In this method, we can add the CSS to a HTML file using the style tag. The style tag is placed in the head section of the HTML file.
  3. External CSS: In this method, we can add the CSS to a HTML file using the link tag. The link tag is placed in the head section of the HTML file. The link tag is used to link the HTML file to an external CSS file.
