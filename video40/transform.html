📘 CSS Notes

  • rotate() rotates an element. It can rotate on the X, Y, and Z axes.
  • skew() tilts an element along the X or Y axis.
  • translateX() moves an element horizontally. You can also use translateY() or translate().
  • Multiple Transform Functions: Use more than one transform by writing them in a single line separated by spaces.
  • text-align:center; only centers inline content (like text) horizontally inside an element.
  • display:flex; turns an element into a Flex Container.
  • Flexbox Properties: justify-content, align-items, flex-direction, gap, flex-wrap.
